## Getting Started

### Prerequisites

- Java 25

### Installation

```bash
./gradlew build
```

### Usage

```bash
java -jar build/libs/app.jar
```

### Formatting

```bash
./gradlew spotlessApply
```

## Features

- Input a day of the week (e.g., "Monday", "Mon", etc.) and receive the corresponding schedule for that day (Thursday not included).
- Exit the program by entering "quit".
