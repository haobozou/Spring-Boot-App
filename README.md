## Getting Started

### Prerequisites

- Java 25
- Gradle 9

### Installation

```bash
gradle build
```

### Usage

```bash
java -jar build/libs/app.jar
```

### Formatting

```bash
gradle format
```

## Features

- Input a day of the week (e.g., "Monday", "mon", etc., case-insensitive) and receive the corresponding schedule for that day (Thursday not included).
- Exit the program by entering "quit".
